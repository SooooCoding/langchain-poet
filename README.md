# LangChain Poet

Learning LangChain by building a poem generator using OpenAI and LangChain.

**Work in Progress**

## Current Features

- Connect to OpenAI using ChatOpenAI
- Load API key with python-dotenv
- Generate poems from a given topic

## Tech Stack

- Python
- LangChain
- LangChain-OpenAI
- OpenAI API
- python-dotenv
